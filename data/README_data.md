# Data Folder Structure

This folder stores local external context for BYOA Study Agent.

## Categories

- course: course-related notes about BYOA, prompting, RAG, MCP, AI IDEs, and agent workflows.
- security: security testing notes, prompt injection notes, and OWASP-style security notes.
- software: software engineering, code review, DevOps, testing, Git workflow, and database notes.
- official: official or semi-official technical notes such as Python version notes.
- csv: structured CSV datasets used by the CSV analysis tool.

The agent scans this folder recursively, so files can be placed inside nested subfolders.
