# AI DevOps and AI SRE

DevOps and SRE focus on running software systems reliably in production. AI can assist with monitoring, incident diagnosis, and remediation suggestions.

## Four Golden Signals

The four golden signals are:

1. Latency: how long requests take.
2. Traffic: how much demand the system receives.
3. Errors: the rate of failed requests.
4. Saturation: how full or overloaded system resources are.

## Incident Response Playbook

A typical incident response process includes:

1. Acknowledge and assess the alert.
2. Check application and database dashboards.
3. Identify recent changes.
4. Localize the blast radius.
5. Apply mitigations.
6. Stabilize and monitor.
7. Communicate progress.
8. Document the root cause and follow-up actions.

## AI SRE

An AI SRE can help map dependencies, summarize observability data, identify likely root causes, and recommend remediation steps. Explainability and auditability are important.
