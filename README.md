## Project Introduction / 项目简介

**BYOA Study Agent** is a local-first AI learning assistant with tool use, local document search, CSV analysis, web fallback, and Markdown note generation.

**BYOA Study Agent** 是一个本地资料优先的 AI 学习助手，支持工具调用、本地资料检索、CSV 分析、网络兜底和 Markdown 笔记生成。
